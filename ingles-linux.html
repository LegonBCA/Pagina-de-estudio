Linux

Lectura Bilingüe

Inglés:

From: Brandon Peters
To: Shirley Tomkins
Subject: Switching to Linux

Español:

De: Brandon Peters
Para: Shirley Tomkins
Asunto: Cambio a Linux

Inglés:
Mrs. Tomkins,
Over in the IT department we've received complaints about the office computers. The hardware itself is fine. The problem, I believe, is the operating system. The system is rife with communication errors and virus threats. Then, there's the fortune we spend on upgrading our software every year. That's why I'm suggesting we switch to Linux® or some other OS related to Unix®. For example, Ubuntu® is very popular with users and works with any hardware including both "Macs" and PCs.

Español:
Sra. Tomkins,
En el departamento de TI hemos recibido quejas sobre las computadoras de la oficina. El hardware en sí está bien. El problema, creo, es el sistema operativo. El sistema está lleno de errores de comunicación y amenazas de virus. Además, está la fortuna que gastamos en actualizar nuestro software cada año. Por eso sugiero que cambiemos a Linux® o algún otro sistema operativo relacionado con Unix®. Por ejemplo, Ubuntu® es muy popular entre los usuarios y funciona con cualquier hardware, incluyendo tanto "Macs" como PCs.

Inglés:
Linux® is free, open source and many of its programs have a free software license. We can get Linux® distributions from vendors that have entire software packages that we can use. For example, OpenOffice® does everything that Microsoft Office® does.

Español:
Linux® es gratuito, de código abierto y muchos de sus programas tienen una licencia de software libre. Podemos obtener distribuciones de Linux® de proveedores que tienen paquetes completos de software que podemos usar. Por ejemplo, OpenOffice® hace todo lo que hace Microsoft Office®.

Inglés:
I've talked to many staff members about this already. Some expressed concern with the interface. Specifically, most people talked with think Linux® required entering commands lines in a terminal. Actually, many Linux® operating systems use a graphical interface just like the average desktop operating system. As for networking, Linux® comes with a program called Samba®. This program allows Linux® to interact with Windows®-based clients. This means we don't have to worry about printer troubles over the network once the TCP/IP protocol is reconfigured.

Español:
Ya he hablado con muchos miembros del personal sobre esto. Algunos expresaron preocupación por la interfaz. Específicamente, la mayoría de las personas con las que hablé piensan que Linux® requiere ingresar líneas de comandos en una terminal. En realidad, muchos sistemas operativos Linux® utilizan una interfaz gráfica igual que el sistema operativo de escritorio promedio. En cuanto a la red, Linux® viene con un programa llamado Samba®. Este programa permite que Linux® interactúe con clientes basados en Windows®. Esto significa que no tenemos que preocuparnos por problemas de impresora en la red una vez que se reconfigure el protocolo TCP/IP.

Inglés:
Please let me know if you have any follow up questions.
Brandon Peters

Español:
Por favor, hágame saber si tiene alguna pregunta adicional.
Brandon Peters

Reading Comprehension

Linux is a popular open-source operating system used in servers, desktops, and embedded systems. It is known for its stability, security, and flexibility. There are many distributions of Linux, each with unique features and package managers.
